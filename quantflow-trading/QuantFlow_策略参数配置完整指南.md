## 1. 均线交叉策略 (MA_CROSS)

### 1.1 策略原理

均线交叉策略是最经典的技术分析策略之一，通过观察短期移动平均线和长期移动平均线的交叉来判断买卖时机。

**核心逻辑**：
- **金叉（买入信号）**：短期均线向上穿过长期均线
- **死叉（卖出信号）**：短期均线向下穿过长期均线

**适用场景**：
- ✅ 趋势明确的市场
- ✅ 波段交易
- ❌ 横盘震荡市场（容易产生假信号）

---

### 1.2 参数格式

```
短期周期,长期周期
```

**参数说明**：
- `短期周期`：快速响应价格变化的均线周期，整数类型
- `长期周期`：平滑价格波动的均线周期，整数类型
- **约束条件**：长期周期必须大于短期周期

---

### 1.3 推荐配置

| 配置名称 | 参数值 | K线周期 | 适用场景 | 交易频率 | 风险等级 |
|---------|-------|---------|---------|---------|---------|
| **超短线** | `5,10` | 5m-15m | 日内交易，快速进出 | 极高 | ⚠️⚠️⚠️ 高 |
| **短线经典** | `5,20` | 15m-1h | 短线波段，最常用 | 高 | ⚠️⚠️ 中高 |
| **中线稳健** | `10,30` | 1h-4h | 中期趋势跟踪 | 中等 | ⚠️ 中 |
| **长线保守** | `20,60` | 4h-1d | 长期趋势确认 | 低 | ✅ 低 |
| **超长线** | `50,200` | 1d-1w | 大周期趋势，金叉死叉 | 极低 | ✅ 极低 |

### 1.5 参数选择指南

#### 短期周期选择

| 周期 | 特点 | 适用场景 |
|-----|------|---------|
| 3-5 | 极度敏感，快速响应 | 超短线、套利 |
| 5-10 | 敏感，跟踪短期趋势 | 日内、短线 |
| 10-20 | 平衡，过滤部分噪音 | 波段、中线 |
| 20-50 | 稳健，趋势确认 | 中长线 |

#### 长期周期选择

| 周期 | 特点 | 适用场景 |
|-----|------|---------|
| 10-20 | 短期趋势确认 | 与3-5短周期配合 |
| 20-50 | 中期趋势确认 | 与5-20短周期配合 |
| 50-100 | 长期趋势确认 | 与20-50短周期配合 |
| 100-250 | 超长期趋势 | 与50-100短周期配合 |

#### 周期比例建议

```
长期周期 / 短期周期 = 2 ~ 10
```

**常用比例**：
- `2倍关系`：如 5/10, 10/20（快速响应）
- `4倍关系`：如 5/20, 10/40（经典配置）
- `6倍关系`：如 10/60, 20/120（稳健配置）
- `10倍关系`：如 20/200（极度稳健）

---

### 1.6 优化建议

1. **回测验证**：
   ```
   测试周期：至少3个月历史数据
   评估指标：胜率、收益率、最大回撤
   ```

2. **参数调整**：
   ```
   初始参数：5,20
   如果信号太少：5,15 或 5,10
   如果信号太多：10,30 或 10,40
   ```

3. **不同市场适配**：
   - 高波动市场（如BTC）：可使用较大参数
   - 低波动市场（如稳定币）：使用较小参数

## 2. 布林带策略 (BOLLINGER_BANDS)

### 2.1 策略原理

布林带策略基于统计学原理，认为价格会围绕均值波动，当价格偏离均值过多时会回归。

**布林带构成**：
- **中轨**：N周期移动平均线（MA）
- **上轨**：中轨 + K × 标准差
- **下轨**：中轨 - K × 标准差

**交易信号**：
- **买入信号**：价格触及或跌破下轨后反弹
- **卖出信号**：价格触及或突破上轨

**统计含义**：
- 标准差倍数为2时，理论上95.4%的价格会在布林带内波动
- 标准差倍数为3时，理论上99.7%的价格会在布林带内波动

**适用场景**：
- ✅ 震荡行情，价格在区间内波动
- ✅ 均值回归交易
- ❌ 强势单边趋势（容易被套）

---

### 2.2 参数格式

```
周期,标准差倍数
```

**参数说明**：
- `周期`：计算均线和标准差的K线数量，整数类型
- `标准差倍数`：控制布林带宽度，浮点数类型
- **约束条件**：
  - 周期 > 0，建议 10-50
  - 标准差倍数 > 0，建议 1.5-3

---

### 2.3 推荐配置

| 配置名称 | 参数值 | 带宽特点 | 适用场景 | 信号频率 | 风险等级 |
|---------|-------|---------|---------|---------|---------|
| **敏感型** | `20,1.5` | 窄带 | 高波动市场 | 高 | ⚠️⚠️⚠️ 高 |
| **标准型** | `20,2` | 标准带 | 通用配置 | 中等 | ⚠️⚠️ 中 |
| **保守型** | `20,2.5` | 宽带 | 低波动市场 | 低 | ⚠️ 低 |
| **短线型** | `10,2` | 标准带 | 短线交易 | 高 | ⚠️⚠️⚠️ 高 |
| **长线型** | `50,2` | 标准带 | 长线交易 | 低 | ⚠️ 低 |

## 3. MACD策略 (MACD)

### 3.1 策略原理

MACD（Moving Average Convergence Divergence，指数平滑异同移动平均线）是最常用的趋势跟踪指标之一。

**计算公式**：
```
DIF = EMA(12) - EMA(26)      // 快线减慢线
DEA = EMA(DIF, 9)            // DIF的9周期EMA
MACD = 2 × (DIF - DEA)       // 柱状图
```

**交易信号**：
- **金叉（买入）**：DIF向上穿过DEA
- **死叉（卖出）**：DIF向下穿过DEA

**辅助判断**：
- MACD柱状图从负转正：买入力量增强
- MACD柱状图从正转负：卖出力量增强
- DIF和DEA都在零轴上方：多头市场
- DIF和DEA都在零轴下方：空头市场

**适用场景**：
- ✅ 趋势明确的市场
- ✅ 中长期趋势跟踪
- ❌ 横盘震荡市场（频繁假信号）

---

### 3.2 参数格式

```
快线周期,慢线周期,信号线周期
```

**参数说明**：
- `快线周期`：短期EMA周期，整数类型
- `慢线周期`：长期EMA周期，整数类型
- `信号线周期`：DIF的EMA周期，整数类型
- **约束条件**：
  - 慢线周期 > 快线周期
  - 建议：慢线 ≈ 快线 × 2
  - 信号线通常为 5-10

---

### 3.3 推荐配置

| 配置名称 | 参数值 | 响应速度 | 适用场景 | 信号频率 | 风险等级 |
|---------|-------|---------|---------|---------|---------|
| **超快型** | `5,10,5` | 极快 | 日内超短线 | 极高 | ⚠️⚠️⚠️ 高 |
| **快速型** | `6,13,5` | 快 | 短线交易 | 高 | ⚠️⚠️⚠️ 高 |
| **标准型** | `12,26,9` | 标准 | 通用配置 ⭐ | 中等 | ⚠️⚠️ 中 |
| **稳健型** | `19,39,9` | 慢 | 中长线 | 低 | ⚠️ 低 |
| **周线型** | `12,26,9` | 标准 | 周线级别 | 极低 | ✅ 极低 |

## 4. RSI策略 (RSI)

### 4.1 策略原理

RSI（Relative Strength Index，相对强弱指标）是用于衡量市场超买超卖状态的震荡指标。

**计算公式**：
```
RS = 平均上涨幅度 / 平均下跌幅度
RSI = 100 - (100 / (1 + RS))
```

**RSI取值范围**：0-100

**市场状态判断**：
- `RSI > 70`：超买区域，价格可能回调
- `RSI < 30`：超卖区域，价格可能反弹
- `RSI = 50`：市场中性

**交易信号**：
- **买入信号**：RSI < 超卖阈值（如30）
- **卖出信号**：RSI > 超买阈值（如70）

**适用场景**：
- ✅ 震荡市场，价格在区间波动
- ✅ 寻找超买超卖机会
- ❌ 强趋势市场（RSI可能长期处于极值区）

---

### 4.2 参数格式

```
周期,超卖阈值,超买阈值
```

**参数说明**：
- `周期`：计算RSI的K线数量，整数类型
- `超卖阈值`：买入信号触发值，浮点数类型
- `超买阈值`：卖出信号触发值，浮点数类型
- **约束条件**：
  - 周期 > 0，建议 7-21
  - 0 < 超卖阈值 < 超买阈值 < 100
  - 典型值：超卖20-40，超买60-80

---

## 5. 网格交易策略 (GRID)

### 5.1 策略原理

网格交易是一种在价格区间内进行高抛低吸的机械化交易策略。

**核心思想**：
- 在价格区间内设置多个等距的买卖价格点（网格）
- 价格下跌到网格线时买入
- 价格上涨到网格线时卖出
- 通过频繁的小额交易积累利润

**工作流程**：
```
1. 设定价格区间：[下界, 上界]
2. 划分N个网格
3. 网格间距 = (上界 - 下界) / N
4. 价格下跌穿过网格线 → 买入
5. 价格上涨穿过网格线 → 卖出
```

**适用场景**：
- ✅ 震荡行情，价格在区间内反复波动
- ✅ 不需要预测价格方向
- ✅ 适合自动化交易
- ❌ 单边趋势市场（容易被套或踏空）

---

### 5.2 参数格式

```
价格下界,价格上界,网格数量
```

**参数说明**：
- `价格下界`：网格区间的最低价格，浮点数类型
- `价格上界`：网格区间的最高价格，浮点数类型
- `网格数量`：将区间分成多少个网格，整数类型
- **约束条件**：
  - 上界 > 下界
  - 网格数量：5-50（建议10-20）
  - 网格间距 = (上界 - 下界) / 网格数量

### 5.5 网格设置指南

#### 确定价格区间

**方法1：技术分析法**
```
1. 查看历史K线图
2. 找出明显的支撑位（下界）
3. 找出明显的阻力位（上界）
4. 确保有足够的历史数据支持

示例（BTC）：
- 近3个月低点：38,000
- 近3个月高点：48,000
- 建议区间：38,000 - 48,000
```

**方法2：波动率法**
```
基于历史波动率：
1. 计算近期日均波动率（如5%）
2. 下界 = 当前价 × (1 - 波动率 × N)
3. 上界 = 当前价 × (1 + 波动率 × N)

示例（当前价40,000）：
- 日波动率：5%
- N取值：3-5倍
- 下界：40,000 × (1 - 0.05 × 3) = 34,000
- 上界：40,000 × (1 + 0.05 × 3) = 46,000
```

#### 计算网格数量

**建议网格间距**：

| 市场类型 | 建议间距 | 网格数量计算 |
|---------|---------|-------------|
| 超低波动（< 1%/天） | 0.5-1% | 区间幅度 / 0.5% |
| 低波动（1-2%/天） | 1-2% | 区间幅度 / 1% |
| 中等波动（2-5%/天） | 2-4% | 区间幅度 / 3% |
| 高波动（> 5%/天） | 5-10% | 区间幅度 / 7% |

## 6. 突破策略 (BREAKOUT)

### 6.1 策略原理

突破策略基于这样的市场规律：价格突破关键阻力或支撑位后，通常会沿突破方向继续运行一段距离。

**核心概念**：
- **阻力位**：价格难以突破的上方价格区域
- **支撑位**：价格难以跌破的下方价格区域
- **突破**：价格突破阻力或支撑位

**突破类型**：
```
1. 向上突破（看涨）：
   - 价格突破近期最高价
   - 成交量放大
   - 表示上涨动能强
   → 买入信号

2. 向下突破（看跌）：
   - 价格跌破近期最低价
   - 成交量可能放大
   - 表示下跌动能强
   → 卖出信号
```

**适用场景**：
- ✅ 趋势行情，强势突破后持续运行
- ✅ 突破关键整数关口
- ✅ 突破前期高点/低点
- ❌ 震荡行情（假突破多）
- ❌ 流动性差的市场

---

### 6.2 参数格式

```
回看周期
```

**参数说明**：
- `回看周期`：观察多少根K线的最高价和最低价，整数类型
- **约束条件**：
  - 周期 > 0
  - 建议范围：10-250
  - 周期越大，信号越少但越可靠

---

|---------|-------|---------|---------|---------|---------|
| **超短线** | `10` | 10周期 | 极高 | 日内突破 | ⚠️⚠️⚠️ 高 |
| **短线型** | `20` | 20周期 | 高 | 短期趋势 ⭐ | ⚠️⚠️ 中高 |

## 8. 参数优化指南

### 8.1 优化流程

```
步骤1：从标准配置开始
- 使用推荐的标准参数
- 运行1-3个月

步骤2：收集数据
- 记录所有交易
- 统计胜率、收益率、回撤

步骤3：分析问题
- 信号太少？→ 缩小周期
- 信号太多？→ 加大周期
- 胜率太低？→ 调整阈值
- 回撤太大？→ 加严止损

步骤4：小幅调整
- 一次只调整一个参数
- 调整幅度 10-20%
- 重新测试

步骤5：回测验证
- 至少测试3个月历史数据
- 包含牛市、熊市、震荡市
- 确保稳定盈利
```

---

### 8.2 评估指标

**核心指标**：
```
1. 总收益率 > 10%（月）
2. 胜率 > 40%
3. 盈亏比 > 1.5
4. 最大回撤 < 20%
5. 夏普比率 > 1
```

**辅助指标**：
```
6. 交易次数：适中，不过频
7. 平均持仓时间：符合预期
8. 连续亏损次数 < 5
9. 最大单笔亏损 < 5%
10. 手续费占比 < 10%
```

---

### 8.3 回测注意事项

```
⚠️ 避免过度优化（过拟合）：
- 参数不是越复杂越好
- 在训练数据上表现完美的参数，实盘可能失效
- 保持策略简单

⚠️ 样本要充分：
- 至少3个月以上数据
- 包含不同市场环境
- 测试多个币种

⚠️ 考虑实际交易成本：
- 手续费
- 滑点
- 资金占用成本

⚠️ 定期重新评估：
- 每季度回顾
- 市场环境变化时调整
- 不要固守失效的参数
```
